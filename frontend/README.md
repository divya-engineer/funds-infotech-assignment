# Mini ERP + CRM Operations Portal

## Project Overview
A full-stack ERP CRM portal for managing customers, products, inventory and sales challans.

## Tech Stack

Frontend:
- React
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB

### Authentication
- User registration
- User login
- Role support

### Customer CRM
- Add customer
- View customers
- Update customer

### Product Management
- Add products
- Manage stock

### Sales Challan
- Create challan
- Reduce stock on confirmation
